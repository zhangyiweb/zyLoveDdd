# zyLoveDdd 💕

一个全屏动画恩爱表白页面，献给最爱的家人。

## 在线预览

部署 GitHub Pages 后访问：

**https://zhangyiweb.github.io/zyLoveDdd/**

## 页面特色

- 全屏沉浸式星空背景，星星闪烁飘落
- 渐变流光标题与跳动爱心动画
- 鼠标粉色光点拖尾，随机飘散小爱心
- 点击屏幕绽放爱心粒子
- 玻璃质感表白卡片，依次滑入展示

## 项目结构

```
zyLoveDdd/
├── docs/
│   └── index.html    # 表白页面（GitHub Pages 入口）
├── .gitignore
└── README.md
```

## 本地预览

直接用浏览器打开 `docs/index.html` 即可，无需安装依赖。

## 部署 GitHub Pages

1. 将代码推送到 GitHub 仓库
2. 进入仓库 **Settings → Pages**
3. **Source** 选择 `Deploy from a branch`
4. **Branch** 选择 `main`，**Folder** 选择 `/docs`
5. 保存后等待 1～3 分钟，即可通过上方链接访问

## 技术说明

- 纯 HTML + CSS + JavaScript，零依赖
- Canvas 实现星空背景与鼠标拖尾
- CSS 动画实现标题、爱心、卡片等动效
- 响应式布局，适配手机与电脑
